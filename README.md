# ResuCraft

ResuCraft is a modern resume optimization platform designed to help job seekers **beat the ATS, impress recruiters, and own their career**.  
It provides instant resume analysis, keyword matching, ATS scoring, and actionable suggestions to level up your resume, and land the interviews.

---

## ‚ú® Features

- **Upload & Manage Resumes**

  - Upload PDF resumes with instant preview.
  - Maintain a personal library of multiple resumes.
  - Delete or add resumes seamlessly.

- **Job-Aware Resume Analysis**

  - Paste any job description and get an instant analysis.
  - See how well your resume aligns with the role.

- **ATS Score Breakdown**

  - Clear ATS score out of 100.
  - Understand how your resume performs against automated screening systems.

- **Keyword Matching**

  - View matched and missing keywords.
  - Get actionable suggestions to improve relevance.

- **Updated Resume**

  - Download an updated version with improvements applied.
  - Use it directly or as a reference for edits.

- **Analysis Archive**
  - Every analysis is saved.
  - Revisit past roles, compare performance, and track your resume‚Äôs evolution.
